UMLS Semantic Network is part of the U.S. National Library of Medicine (NLM) Unified Medical Language System (UMLS). It contains:

  • Semantic Types, a set of broad subject categories that provide a consistent categorization of all concepts represented in the UMLS Metathesaurus®, and
  • Semantic Relations, a set of useful and important relationships that exist between Semantic Types.

Source Home Page: http://semanticnetwork.nlm.nih.gov

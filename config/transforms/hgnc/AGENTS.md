# Scope

This directory contains HGNC transform property files retained with the operations package.

# HGNC Transform Property Files

The property names in these files are expected to match HGNC download column names with spaces replaced by underscores. Keep property names aligned with upstream HGNC headers.

# Delimiter And Property Mapping Rules

- `DelimitedColumns.properties` identifies columns that contain multiple values.
- `Delimiters.properties` describes the delimiter mode used for those columns.
- `PropertyMap.properties` maps HGNC columns to output properties.
- `SpecialistDatabase.properties` lists specialist database identifiers.

# Core Files

- `configReadMe.txt`: local reference for delimiter codes and HGNC header alignment.
